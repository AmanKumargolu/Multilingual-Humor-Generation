# Multilingual Humor Generation System using LLMs

This project focuses on the design and implementation of a multilingual humor generation system using Large Language Models (LLMs).

The system is capable of generating humorous text in multiple languages by applying techniques from Artificial Intelligence (AI), Natural Language Processing (NLP), and Machine Learning (ML).

This project was developed during my Winter Internship at NIT Patna under academic guidance.

- Natural Language Processing (NLP)
- Machine Learning
- Large Language Models (LLMs)
